# Estética musical del proyecto

## 1. Dirección general

El proyecto no busca producir sonidos meramente ilustrativos. La meta es construir una poética sonora de la topología.

La música debe conservar una relación con la estructura del nudo, pero también debe tener vida musical propia.

## 2. Piano como voz topológica

Una decisión inicial importante es que el piano puede llevar la estructura principal. El piano funciona como una voz clara, articulada y suficientemente flexible para representar:

- patrones;
- repeticiones;
- inversiones;
- simetrías;
- tensiones;
- registros;
- motivos.

Después pueden integrarse otros instrumentos como bajo, guitarra, percusión, cuerdas o sintetizadores.

## 3. Jazz como solución al caos

El jazz aparece como un lenguaje natural para este proyecto porque permite combinar estructura e improvisación.

Un nudo tiene restricciones; el jazz también.  
Un nudo permite deformaciones; el jazz también.  
Un nudo conserva identidad bajo transformaciones; una improvisación también puede conservar un motivo bajo variaciones.

## 4. Conversación instrumental

La sonificación puede pensarse como conversación:

- piano: voz topológica principal;
- bajo: soporte geométrico o trayectoria;
- percusión: cruces, acentos, cambios de orientación;
- guitarra u otro instrumento: respuesta, variación o sombra armónica.

## 5. Referencia estética

Una referencia importante es *A Love Supreme*, especialmente por su carácter modal, espiritual, repetitivo, expansivo e improvisatorio.

No se trata de copiar esa obra, sino de tomarla como inspiración para pensar cómo una célula simple puede abrir un espacio musical profundo.

## 6. Principio compositivo

```text
nudo → motivo → repetición → variación → diálogo → forma
```

## 7. Evitar dos extremos

Debe evitarse:

1. Que la música sea bonita pero matemáticamente arbitraria.
2. Que la sonificación sea matemáticamente fiel pero musicalmente inerte.

El proyecto debe buscar una tercera vía: rigor estructural y potencia estética.
