# 08. Difusión y álbum de sonificaciones

Una meta del proyecto es que los resultados no se queden solamente como código, sino que puedan escucharse y compartirse.

## Producto de difusión

Se propone una colección de piezas sonoras breves, cada una asociada a una estructura matemática.

Ejemplo:

| Pista | Objeto matemático | Mapeo principal | Salida |
|---|---|---|---|
| 01 | Nudo trébol | Cruces / palabra de trenza | WAV / Sonic Pi |
| 02 | Nudo figura ocho | Secuencia de cruces | WAV / MIDI |
| 03 | Nudo toral | Parámetros p, q | WAV / visualización |
| 04 | Familia pretzel | Parámetros de familia | WAV |
| 05 | Invariante theta | Coeficientes | WAV / pieza extendida |

## Plataformas posibles

- GitHub: documentación, código y resultados reproducibles.
- YouTube: videos con visualización + audio + explicación.
- SoundCloud: publicación de piezas sonoras.
- Spotify: publicación posterior si se decide construir un álbum formal.
- GitHub Pages: página sencilla del proyecto.

## Formato recomendado de cada pieza

Cada ejemplo sonoro debería acompañarse de:

1. nombre del objeto matemático;
2. representación usada;
3. regla de mapeo;
4. código que genera el audio;
5. archivo sonoro;
6. figura o animación;
7. explicación breve para público general;
8. explicación técnica para público académico.

## Nombre posible de colección

- Nudos sonoros
- Topologías audibles
- Música de nudos
- Sonidos de la forma
- Cómo suena un nudo
- Sonorización de estructuras matemáticas
