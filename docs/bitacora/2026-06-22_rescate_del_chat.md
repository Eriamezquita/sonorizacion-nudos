# Bitácora: rescate inicial del chat

Fecha: 2026-06-22

## Situación

El proyecto ha avanzado mucho en conversación, pero el repositorio todavía no refleja ese avance. La preocupación principal es no perder las ideas desarrolladas en ChatGPT.

## Decisión

Crear una estructura documental amplia en GitHub antes de seguir acumulando código suelto.

## Ideas que deben preservarse

- El proyecto no es solamente técnico: también es conceptual, musical y artístico.
- El piano puede ser la voz principal del nudo.
- El jazz puede funcionar como lenguaje para integrar estructura e improvisación.
- Un nudo no debe entenderse como una canción cerrada, sino como un espacio de posibilidades.
- Sonic Pi será el primer motor sonoro accesible.
- Python servirá para generar datos, mapeos y visualizaciones.
- Theta invariant queda como una línea futura, no como primer paso obligatorio.

## Próximo paso

Subir al repositorio los documentos base y después empezar a reemplazar los esbozos por versiones más rigurosas.
