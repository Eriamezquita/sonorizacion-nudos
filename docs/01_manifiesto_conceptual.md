# Manifiesto conceptual

## Punto de partida

La sonorización de nudos no debe entenderse como una traducción mecánica donde cada nudo produce una melodía fija. El objetivo es más profundo: construir un sistema donde la estructura topológica genere condiciones musicales.

Un nudo puede pensarse como una forma de memoria espacial: una curva cerrada que se entrelaza consigo misma sin romperse. Al sonificarlo, no se busca "decorar" la matemática con sonido, sino hacer audible una parte de su organización interna.

## Tesis conceptual inicial

Un nudo no genera una canción.  
Un nudo genera un campo de posibilidades.

Ese campo puede convertirse en:

- motivo melódico;
- patrón rítmico;
- textura;
- gesto;
- forma;
- restricción de improvisación;
- diálogo entre instrumentos.

## Capas del proyecto

```text
1. Capa matemática
   Nudo, diagrama, cruces, orientación, invariantes, familia.

2. Capa computacional
   Datos, coordenadas, representaciones, listas, matrices, funciones.

3. Capa sonora
   Altura, duración, ritmo, timbre, amplitud, densidad, espacialidad.

4. Capa musical
   Motivo, frase, repetición, variación, tensión, resolución.

5. Capa compositiva
   Forma, improvisación, conversación instrumental, arreglo.
```

## Pregunta guía

¿Cómo puede escucharse la diferencia entre dos nudos?

Esta pregunta no se limita a generar sonidos distintos. Exige diseñar criterios para que las diferencias matemáticas produzcan diferencias perceptibles y musicalmente significativas.

## Principio artístico

La matemática aporta estructura.  
La música aporta experiencia temporal.  
La sonificación es el puente entre ambas.
