# Plan de trabajo para servicio social y tesis

## Eje general

Desarrollar herramientas conceptuales y computacionales para sonorizar estructuras de la teoría de nudos.

## Fase I: instalación, capacitación y base técnica

Objetivos:

- Instalar y aprender herramientas básicas.
- Usar Python para datos, visualización y generación de secuencias.
- Usar Sonic Pi para escuchar prototipos.
- Introducir conceptos de sonificación.
- Introducir teoría de nudos.

Productos:

- repositorio organizado;
- primeros scripts;
- primeras pruebas en Sonic Pi;
- documentación inicial.

## Fase II: diseño de mapeos y prototipos

Objetivos:

- Diseñar estrategias matemático-sonoras.
- Trabajar con nudos básicos.
- Generar ejemplos audibles.
- Comparar mapeos.
- Documentar decisiones.

Productos:

- ejemplos del nudo trébol;
- ejemplos del nudo figura ocho;
- primeros mapeos formales;
- bitácora de escucha;
- visualizaciones.

## Fase III: ampliación, documentación y difusión

Objetivos:

- Extender a familias de nudos.
- Incorporar invariantes más complejos.
- Generar material sonoro organizado.
- Preparar informe final.
- Ordenar el repositorio para difusión.

Productos:

- informe de servicio social;
- base para tesis;
- audios;
- documentación;
- posibles notebooks explicativos;
- presentación del proyecto.

## Horizonte temporal

La ruta general puede extenderse hasta mayo de 2027, considerando una evolución desde prototipos básicos hasta una versión documentada y defendible académicamente.
