# Tabla de mapeos v0

| Rasgo del nudo | Parámetro sonoro | Comentario |
|---|---|---|
| Cruce | Ataque o evento | Cada cruce puede producir una nota o golpe |
| Cruce positivo | Ascenso / brillo | Puede asociarse a nota ascendente, timbre claro o acento |
| Cruce negativo | Descenso / sombra | Puede asociarse a nota descendente, timbre oscuro o registro bajo |
| Paso por arriba | Registro agudo | Diferencia audible simple |
| Paso por abajo | Registro grave | Complemento del paso por arriba |
| Número de cruces | Densidad | Más cruces, más eventos |
| Orientación | Dirección melódica | Puede orientar la frase |
| Curvatura | Intensidad / vibrato | Mayor curvatura, mayor tensión |
| Longitud de arco | Duración | Relación entre recorrido y tiempo |
| Invariante | Motivo persistente | Rasgo estable bajo transformaciones |
| Familia de nudos | Modo o escala | Cada familia puede tener paleta propia |
