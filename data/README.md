# Carpeta de datos

- `raw/`: datos originales sin modificar.
- `processed/`: datos transformados o normalizados para sonificación.

Ejemplos futuros:

- representaciones PD de nudos;
- palabras de trenza;
- coeficientes de polinomios;
- coeficientes del invariante theta;
- tablas de parámetros sonoros.
